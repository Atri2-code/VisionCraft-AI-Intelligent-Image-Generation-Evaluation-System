# VisionCraft-AI-Intelligent-Image-Generation-Evaluation-System


An end-to-end Generative AI system that enhances prompts, generates multiple images, evaluates them using CLIP, and selects the best output with feedback.

---

## 🚀 Features
- ✨ Prompt enhancement for better image quality  
- 🖼️ Image generation using Stable Diffusion  
- 📊 CLIP-based image-text evaluation  
- 🏆 Automatic ranking and best image selection  
- 🎛️ Interactive UI using Gradio  

---

## 🧠 How It Works
1. User inputs a prompt  
2. Prompt is enhanced for quality and detail  
3. Multiple images are generated  
4. CLIP evaluates image-text alignment  
5. Best image is selected and shown with feedback  

---

## ⚙️ Tech Stack
- Python  
- PyTorch  
- HuggingFace Diffusers  
- CLIP (by OpenAI)  
- Gradio
